# Nepal Flood Risk Intelligence System

Final Year BCA Project

## Overview

A district-level flood risk prediction and monitoring system for Nepal.

The system combines:

- NASA POWER weather data
- GloFAS river discharge data
- BIPAD flood incidents
- BIPAD heavy rainfall incidents
- District elevation data

Machine Learning:

1. Linear Regression
   - Predicts Flood Impact Index (FII)

2. Decision Tree
   - Predicts Risk Level
   - LOW / MEDIUM / HIGH

Technology Stack:

- Python
- PostgreSQL
- Docker
- Apache Airflow
- FastAPI
- Plotly Dash
